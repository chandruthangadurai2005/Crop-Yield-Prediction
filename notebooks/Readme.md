This project focuses on predicting crop yield using historical agricultural, climatic, and input-related data. Multiple datasets were collected and merged, followed by preprocessing and feature engineering to prepare the data for regression-based machine learning models.





## Crop Yield Prediction (SIH25044)

This project focuses on preprocessing and preparing multi-source agricultural data
for building a crop yield prediction and optimization system.

Current stage:
- Data cleaning
- Yield validation
- Feature encoding



### Data Preprocessing
- Cleaned and validated multi-source agricultural datasets
- Recomputed crop yield from production and area
- Handled missing climatic data using state-wise statistics
- Removed outliers using IQR method
- Encoded categorical features for regression models
